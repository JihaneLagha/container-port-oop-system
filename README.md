# Container Port OOP System

## Overview
This is a Java Object-Oriented Programming project that simulates a port management system with ships, ports, containers, and container types.

## Features
- Manage ships and ports
- Support different container types
- Use object-oriented concepts such as inheritance, interfaces, and polymorphism
- Include test cases and expected outputs

## Technologies Used
- Java
- Object-Oriented Programming
- Eclipse IDE
